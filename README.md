### Git Commands

1. `git init`
2. `git add <file>` OR `git add .`
3. `git commit -m "YOUR MESSAGE"`
4. `git add REMOTE_NAME REMOTE_URL`
    - `REMOTE_NAME` will usually be `origin`
5. `git push REMOTE_NAME BRANCH_NAME`
